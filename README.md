### 系统环境

* Ubuntu 20.04 LTS
* Postgres 12.x
* Python 3.11.x

### Web Framework

* *django-4.2.* [相关文档](https://docs.djangoproject.com/en/4.2/)

### Build

```
pip install -r requirements/local.txt  # development

pip install -r requirements/prod.txt # production
```
