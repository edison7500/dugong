### 系统环境
* Ubuntu 16.04
* Nginx
* MySQL
* Python3.6
* Nodejs


### Web Framework
* *django-1.11* [相关文档](https://docs.djangoproject.com/en/1.11/)


### 前端框架
* *bulma* [相关文档](https://bulma.io/)


### Build
```
pip install -r requirements/local.txt  # development

pip install -r requirements/production.txt # production
```
